Edu - Logo package
Files included:
- logo.svg          : base SVG (200x200 viewBox)
- logo-dark.svg     : dark background SVG variant
- logo-light.svg    : light background SVG variant
- logo-512.png      : 512x512 PNG (use for PWA manifest / mobile icons)
- logo-512-dark.png : 512x512 PNG on dark background
- logo-512-light.png: 512x512 PNG on light background
- logo-social-1024.png: 1024x1024 PNG suitable for social avatars/covers
- favicon.ico       : ICO containing 16x16 and 32x32 icons

Usage notes:
- SVGs are the source of truth; use them for scaling.
- PNGs are rasterized approximations for immediate use.
